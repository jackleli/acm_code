# Trick记录
## 区间问题（贪心）

### 区间选点
题意：给定 $n$ 个区间，求最少选择几个点，使得每个区间都至少包含一个选出的点

trick：区间按右端点从小到大排序，记录最后一个选择的点为 $x$，如果当前区间 $l\leq x$，则跳过该区间，否则选择新的点为当前区间的右端点，即令 $x\to r$

### 最大不相交区间数量
题意：给定 $N$ 个区间，请你在数轴上选择若干区间，使得选中的区间之间互不相交（包括端点），求选取区间的最大数量

trick：区间按右端点从小到大排序，每次能选的话就选区间的右端点，记最后一个选择的区间右端点为 $x$，如果当前区间 $l\leq x$，则跳过该区间，选择当前区间的右端点，即令 $x\to r$

### 区间分组
题意：给定 $N$ 个区间，请你将这些区间分成若干组，使得每组内部的区间两两之间（包括端点）没有交集，求最小分组数量

trick：区间按左端点从小到大排序，维护所有分组集合区间的右端点，对于当前区间 $[l,r]$ 找到所有分组里面右端点最小的 $r_{\min}$, 如果 $r_{\min}$ < $l$, 说明当前区间可以加入分组里面，否则为新分组


### 区间覆盖
题意：给定 $N$ 个区间，以及一个区间 $[s,t]$, 求最少选择区间数，将指定区间完全覆盖，如果无法完全覆盖输出 $-1$

trick：区间按左端点从小到大排序，在所有能覆盖 $l \leq t$ 的区间中，选择右端点最大的区间，然后更新 $st\to r_{\max}$
